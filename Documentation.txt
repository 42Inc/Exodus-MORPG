
ServerEngine - Класс сервера
  ip - 127.0.0.1 (пока дефолт)
  Порт чтения - 65501 (пока дефолт)
  Порт записи - 65502 (пока дефолт)
  run_server - Запуск сервера на ip и двух смежных портах (один на ввод, другой на вывод)
  wait_clients - Ожидание клиентов. Отправляет подключение в новый процесс. Процесс detach
  get_readline_socket - Получает строку, пришедшую на сокет. Если очередь сообщений пуста - "". На время выполнения блокирует @readline_socket
  set_readline_socket - Добавляет в очередь строку, пришедшую на сокет. Если очередь сообщений пуста - "". На время выполнения блокирует @readline_socket
  client - Функция работы с клиентом. Главная функция дочерних процессов
  sign_in - Функция входа в учетку (Дописать работу с паролем)
  sign_up - Функция регистрации учетки (Дописать работу с паролем)
  disconnect_client - Отрубает клиента от сервака
  shutdown - Отрубает сервак
  reply_err - Ответить с [Wrong Data]
  send_to_client - Отправить строку клиенту
  create_wait_clients_thread - Создать поток, ожидающий клиентов
  create_reader_thread - Создать поток, читающий сокет от клиента

ClientEngine - Класс клиента
  connect - Подключается к серваку по ip и двум портам
  get_readline_socket - Получает строку, пришедшую на сокет. Если очередь сообщений пуста - "". На время выполнения блокирует @readline_socket
  set_readline_socket - Добавляет в очередь строку, пришедшую на сокет. На время выполнения блокирует @readline_socket
  get_readline_stdin - Получает строку, пришедшую на stdin. Если очередь сообщений пуста - "". На время выполнения блокирует @readline_stdin
  set_readline_stdin - Добавляет в очередь строку, пришедшую на stdin. На время выполнения блокирует @readline_stdin


AdmConsole - Класс для управления сервером из консоли, где он запущен
  adm_get_command - Получает команду из stdin. Возвращает индекс команды из adm_command_handler
  adm_command_handler - Возвращает индекс команды или -1, если команды нет

Database - Класс для раоты с базой данных (Файлы в папке data)
  set_pid -  pid процесса, вызывающего класс
  user_add - Добавляет пользователя. Проверяет уникальность логина. Добавить работу с паролем(Пока онли логин). На время выполнения блокирует файл logins.dat
  send_adm_command - Посылает индекс команды адм-консоли всем процессам через файл, который все чекают постоянно. На время выполнения блокирует файл adm_command.dat
  get_adm_command - Возвращает индекс команды адм-консоли. На время выполнения блокирует файл adm_command.dat
  correct_user? - Чекает есть ли логин в базе(true если есть, иначе - false). На время выполнения блокирует файл logins.dat
  wait_unlock - Ожидает разблокировки файла. (0 в файле .lock)
  lock - Блокирует файл
  unlock - Разблокирует файл. (0 в файле .lock)
  user_del - Удаляет пользователя.Должна быть доступна по паролю из AdmConsole
  Сделать для базы сброс пароля пользователя
